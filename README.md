<details>
<summary><b>ğŸŒ 1. The Easiest Way: Using a Web Browser (No installation needed)</b></summary>




Fork the Repository: Click the Fork button at the top right of this page to create a copy of this repo in your own GitHub account.

Navigate to the Notes folder: Open the Notes directory.

Create your folder: Click Add file -> Create new file.

Name it: Type Your_Name/Note_Title.md (or upload your PDF/Doc).

Commit: Scroll down, add a short description, and click Commit changes.

Pull Request: Go back to the main page of your forked repo, click Contribute -> Open pull request.

Click Create pull request and you're done! ğŸ‰

</details>

<details>
<summary><b>ğŸ§ 2. The Hacker Way: Linux CLI (Arch / Ubuntu)</b></summary>




Open your terminal and run the following commands:

# 1. Clone the repository
git clone [https://github.com/YOUR_USERNAME/STA201.git](https://github.com/YOUR_USERNAME/STA201.git)
cd STA201

# 2. Create a new branch for your notes
git checkout -b adding-my-notes

# 3. Create your folder and add your files
mkdir -p Notes/Your_Name
cp ~/path/to/your/note.pdf Notes/Your_Name/

# 4. Add, commit, and push your changes
git add Notes/Your_Name/
git commit -m "Added notes for lecture 3 by Your_Name"
git push origin adding-my-notes


After pushing, GitHub will give you a link in the terminal. Click that link to create your Pull Request!

</details>

<details>
<summary><b>ğŸªŸ 3. The Desktop Way: Windows 10 (Git Bash)</b></summary>




Download and install Git for Windows.

Open Git Bash from your start menu.

Clone the repo to your Documents:

cd ~/Documents
git clone [https://github.com/YOUR_USERNAME/STA201.git](https://github.com/YOUR_USERNAME/STA201.git)
cd STA201


Create a branch:

git checkout -b my-notes-update


Open the folder in File Explorer, navigate to Notes, create a folder with your name, and paste your files inside.

Go back to Git Bash and sync it to GitHub:

git add .
git commit -m "Uploading my notes"
git push -u origin my-notes-update


Go to GitHub in your browser and click the green Compare & pull request button!

</details>
